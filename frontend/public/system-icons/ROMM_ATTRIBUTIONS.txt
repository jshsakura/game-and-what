ngp.svg (Neo Geo Pocket Color), ws.svg (WonderSwan Color), lynx.svg (Atari
Lynx), pcecd.svg (PC Engine CD; uses the TurboGrafx-16 console art from tg16.svg,
to read distinct from the white PC Engine icon) and videopac.svg (Magnavox
Odyssey² / Philips Videopac; from odyssey-2.svg) are from the RomM project's
platform icon set (rommapp/romm, frontend/assets/platforms), same Libretro/CC BY
4.0 lineage.
  Source:  https://github.com/rommapp/romm

zxs.png (Sinclair ZX Spectrum) and c64.png (Commodore 64) are converted from the
RomM platform icons (zxs.ico / c64.ico), same Libretro/CC BY 4.0 lineage.

chip8.svg (CHIP-8) is an original icon made for this project (no attribution
required).
